# Questadon (Eleventy)

Questadon is a personal Eleventy (11ty) site used as a dev garden / experimentation space for writing, programming notes, and project documentation.

## Tech Stack
- Node.js (tested with Node 18+)
- Eleventy (@11ty/eleventy)

## Setup

Clone the repo:

```bash
git clone https://github.com/sae-br/questadon-11ty.git
cd questadon-11ty
```

Run the local dev server:
```bash
npm start
```

The site will be available at:
http://localhost:8080

## Notes
- node_modules/ and _site/ are generated locally and should not be committed
- This project is typically kept cold and cloned only when actively working on it
